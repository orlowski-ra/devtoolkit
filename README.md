## 🚀 Features

### 1. JSON Formatter & Validator
- Format JSON with proper indentation
- Minify JSON for production
- Real-time syntax error detection
- Syntax highlighting for better readability

### 2. Base64 Encoder/Decoder
- Encode text to Base64
- Decode Base64 to text
- Handles Unicode characters properly

### 3. URL Encoder/Decoder
- Encode URLs and special characters
- Decode encoded URLs
- Perfect for API testing and debugging

### 4. Markdown Preview
- Live preview as you type
- Full GitHub-flavored Markdown support
- Copy generated HTML

### 5. Color Contrast Checker
- WCAG 2.1 AA & AAA compliance checking
- Real-time contrast ratio calculation
- Visual preview of color combinations
- Accessibility-first design validation

- **Pure HTML5** - No frameworks, maximum performance

## 🚀 Getting Started

### Local Development
```bash
git clone https://github.com/yourusername/devtoolkit.git
cd devtoolkit
# Open index.html in your browser
# Or serve with any static server:
python -m http.server 8000
npx serve .
```

### Deployment

#### GitHub Pages (Free)
1. Fork this repository
2. Go to Settings → Pages
3. Select "Deploy from a branch" → "main"
4. Your site will be live at `https://yourusername.github.io/devtoolkit`

#### Netlify (Free)
1. Drag and drop the project folder to [Netlify Drop](https://app.netlify.com/drop)
2. Get an instant live URL
3. Custom domains supported

#### Vercel (Free)
```bash
npm i -g vercel
vercel --prod
```
