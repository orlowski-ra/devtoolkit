## 📝 Product Hunt Launch

### Tagline
Essential developer utilities with zero friction

### Description
DevToolkit is a free, open-source suite of developer utilities that runs entirely in your browser. No accounts, no ads, no tracking - just fast, private tools that work offline.

Features include JSON formatting, Base64 encoding, URL encoding, Markdown preview, and WCAG-compliant color contrast checking. Built with pure HTML/CSS/JS for maximum performance and transparency.

Perfect for quick conversions, debugging, and accessibility testing without sending your data to third-party servers.
