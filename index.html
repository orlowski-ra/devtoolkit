Frequently Asked

Quick answers to common questions

Is DevToolkit really free? ▼

Yes! The hosted version is 100% free and always will be. No ads, no tracking, no upsells. We make money by selling the source code to developers who want to self-host or customize it.

Can I use this commercially? ▼

Absolutely! The source code comes with an MIT License, which means you can use it for personal projects, client work, or even resell it. No attribution required.

Does it work offline? ▼

Yes! Once loaded, all tools work entirely in your browser without any server communication. Perfect for airplanes, spotty WiFi, or privacy-conscious workflows.

What tech stack is used? ▼

Pure HTML, CSS, and vanilla JavaScript. No React, no Vue, no build step. Just clean, fast code that works everywhere and is easy to understand and modify.

How do I deploy this? ▼

Deploy anywhere static sites are supported: GitHub Pages (free), Netlify (free), Vercel (free), Cloudflare Pages (free), or your own server. Full guides included with source code purchase.
